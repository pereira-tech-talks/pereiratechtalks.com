APIs y WebSockets con SailsJs && Monitoreo y visualización de datos con StatsD y Grafana

APIs y WebSockets con SailsJs && Monitoreo y visualización de datos con StatsD y Grafana.

En PereiraJs nuestros Speakers se lucieron hablando sobre contrucción de APIs con SailsJs y Monitoreo con StatsD y Grafana:

  1. APIs y WebSockets con SailsJs by John Morales

Conoceras cómo puedes contruir de manera casí automatizada APIs reactivas mediante el uso de WebSockets con SailsJs.

Slides disponibles en: http://slides.com/johndarwinmoralesgonzalez-1/rocka-presentations-3-5#/

  1. Monitoreo y visualización de datos con StatsD y Grafana by Manuel Pineda

Mediante un par de demos usando microservicios en Docker, Manuel Pineda nos dio una charla muy interesante cuya premisa era “cómo conseguir un aumento”, buscando crear un gran impacto en tu organización al mejorar el monitoreo y visualización de datos mediante StatsD, Graphite y Grafana.

Slides disponibles en: http://pin3da.github.io/slides/monitoring.html#1
